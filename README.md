# Territory (рабочее название)

Медитативная multiplayer .io игра о захвате территории. Этап 1.1 — сетевой скелет.

## Требования

- Node.js 20+
- npm 9+

## Запуск

```bash
npm install          # один раз, из корня репозитория

# В двух отдельных терминалах:
npm run dev:server   # сервер на ws://localhost:8080
npm run dev:client   # клиент на http://localhost:5173
```

Откройте http://localhost:5173 в двух-трёх вкладках — каждая вкладка
получает своё цветное пятно территории, растущее в реальном времени.

## Проверка типов

```bash
npm run typecheck
```

## Структура

```
packages/
├── shared/   # типы, константы, протокол — общие для клиента и сервера
├── server/   # Node.js + ws: авторитетная симуляция, игровой цикл
└── client/   # Vite + PixiJS: рендер и сеть
```

## Что уже есть

- Монорепо (npm workspaces), строгий TypeScript, без шага сборки shared
- Авторитетный сервер: тик 10/с, карта 256×256 (Uint16Array)
- Спавн игрока пятном, авто-расползание территории (frontier-алгоритм)
- JSON-протокол: снапшот (base64) + подельтовые диффы
- Клиент: PixiJS v8, карта как GPU-текстура, частичные обновления
```
